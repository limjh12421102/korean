* 종속적 연결어미: -면, -다가, -도록, -자마자, -지만

    압기팁) 라면 먹다가 죽도록 맞았지만



* 보조적 연결어미: -아, -어, -게, -지, -고

    - (본용언 + 보조용언) 에서 본용언에 붙는다.

    - 본용언: 본래의 뜻(o)

    - 보조용언: 본래의 뜻이(x) 아니게 됨.

    
        ex) 먹어 보다 

            - 1개의 서술어! => 문장도 1개.

            - 띄어씀이 원칙이지만, 붙여씀도 허용함. ( ∵ 하나의 서술어로 보기때문에 )








* 전성: 성질이나 역할을 "잠시" 바꾸어줌 즉(=), 본래의 품사는 변함이 없다!

    - 활용된 것!

    

* 파생: 뜻을 바꾸어주거나 품사를 바꾸어 줌.

    - 새로운 단어가 탄생!


* 어근: 실질적으로 뜻을 가진 부분.



* 명전어 명파접 구분법

    - 주와와 호응이 되면        (o)     =>      명전어
    - 부사의 수식을 받으면      (o)     =>      명전어


    - 관형어의 수식을 받으면    (o)     =>      명파접 
    




* 어간에 받침 "ㄹ" 이 있는 것의 명사 활용형은 "ㄻ"

    ex) 얼다 => 얾 (품사 그대로 동사)



Tip: 계속 입으로 말해야 용어에 익숙해짐.






* 전성어미: 성질을 바꿔주는 어미

    ① 명사형 전성어미       :       명사형태로 잠깐 성질을 바꿔주는 어미

        ex) -음, -ㅁ, -기


    ② 관형사형 전성어미     :       관형사형태로 잠깐 성질을 바꿔주는 어미

        ex) -는, -ㄴ, -ㄹ

    
    ③ 부사형 전성어미       :       부사형태로 잠깐 성질을 바꿔주는 어미

        ex) -게, -도록

    



Tip: 애매해도 절묘하게 잘~ 연결시켜줄테니까 지금 당장은 이해가 안되더라도 참고 강의듣자~ (나중에 쭉~ 다~ 연결됨.)








* 향태소: 뜻 가진말의 최소단위.


   ① 실질성( = 실질적인 뜻이있나? ) 


        (o): ㉠ 실질형태소

            ex) ( 체언, 수식언, 용언의 일부 어간, 감탄사 ) => 어근 ( = 실질형태소 )


        (x): ㉡ 형식형태소

            ex) 조사, 선어말어미 어말어미, 접사





    ② 자립성( = 다른것의 도움을 받지 않고 혼자서 쓸 수 있는가? )



        (o): ㉠ 자립형태소

            ex) 체언, 수식언, 감탄사

        (x): ㉡ 의존형태소

            ex) 조사, 어간( <= 실질성은 있음 ), 선어말어미 어말어미, 접사
