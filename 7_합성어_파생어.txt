* 첩어, 의성어·의태어는 100% 부사



* 단일어: 어근 1개



* 복합어 <= 새로운 말을 만들기 위해 "복합어"를 만든다. ( ∵ 어근을 만들기는 어렵기 때문에 주로 복합어 위주로 신조어를 창조한다. )


    ① 합성어 - 어근(실질형태소) + 어근(실질형태소)

        - by합성법 만들어진 말


        ㉠ 통사적 합성어    - 국어문법, 국어문장구조에 맞는! 합성법으로 만들어진 말/

            ex) 국밥, 날아가다, 힘들다

        ㉡ 비통사적 합성어  - 국어문법, 국어문장구조에 맞지않은! 합성법으로 만들어진 말.

            ex) 등산, 오가다, 보슬비 

                cf) 보슬비를 통사적 합성법으로 만든다면? => 보슬보슬내리는비



    ② 파생어 - 뜻 또는 품사를 바꾸어주는것을 파생이라고 한다.

        - by파생법 만들어진 말


        ㉠ 접두사 + 어근

            - 뜻만 추가 (o)

            - 접두사가 꾸며서 한정해준다해서 한정적 접두사 라고 함.

            
            ex) 새빨갛다    

                - "새"(아주)는 떼어내면 뜻이 사라짐. 

                - 접두사 + 어근 (새빨갛) = 어간


                cf) 새색시 

                    - "새"(new)는 떼어내도 뜻이 유지.

                    - 합성어이다. 통사적합성어

            
                cf) 강마르다

                    - 원래 접두사는 품사를 바꿔주지 못하지만 예외가 있다

                    - 마르다(동사) => 깡마르다(형용사)

                    - 품사를 바꿔주는 접두사는 지배적 접두사 라고 함.

                    - "강"(매우)는 떼어내면 뜻이 사라짐.

                    - 접두사 + 어근 (강마르) = 어간


        
        ㉡ 어근 + 접미사 

            1. 뜻 추가 (o)

            2. 품사 바꿈 (o)

            3. 문장구조를 변화시킴 (o) <= 피동·사동접미사



            ex) 먹다 => 먹히다

                - 품사 변화 (x) 둘다 동사

                - 뜻 추가 (o) <= "히" 피동접미사가 "당함"의 뜻을 추가

                - 어근 + 접미사 (먹히) = 어간

                

            ex) 얼다 => 올음

                
                - 품사 변함 (o) 동사에서 명사로

                - 뜻은 그대로 

                - 음 <= 명파접 (명사로 파생시키는 접미사)
