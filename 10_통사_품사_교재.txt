* 날짐승: 날아다니는 짐승

    - 통사적 합성어
        
        이유는 " 날 + ㄹ + 짐승 " 이니까 관형어 + 명사는 국문법에 맞다.

* 날고기, 날것 등등. 生것의 의미는 "날"을 접두사로봐서 파생어라고 본다.



* 의존명사 구별법

    ① 조사를 넣어본다.

    ② -것, -수, -경우 로 바꿔본다.
   

* 보통은 "관형어 + 의존명사" 인데 "명사 + 의존명사"가 오기도한다.

    ex) 빚 때문에 고생을 하다

    ex) 간 때문이야


* 의존명사 예시

    ex) 저 분이 우리를 살렸다.

    ex) 이곳에 온 지 벌써 ~ <= "지" 는 기간

    ex) 최선을 다할 뿐(따름)이다.

    ex) 시키는 대로 못하겠어.

    ex) 빼다 닮은 듯 보인다.

    ex) 그럴 리가 없어.


* 미지칭 부정칭 3인칭



* 제 3자는 빠져

    - 제(재귀대명사) 3(3인칭)  <= 재귀대명사는 3인칭대명사이다


* 재귀대명사

    ex) 그는 스스로의 힘으로 자립을 했다.

        => 스스로


* 주 예수

    - 주격조사 "에서" <= "에서" 가 주격조사인경우도 있다!


* 보조사 "마다"

    - 균일

        ex) 사람사람마다 다 재능을 가지고 있다.


* 다른(관형사) Vs 다른(형용사의 관형어)

    - 관형사일때는 선택의 의미, 형용사의 관형어는 비교대상과의 차이

        ex) 그거말고 다른 영화 볼래 (another one)

        ex) 너랑 생각이 다른 나 (차이)

    

* 어떤(관형사) Vs 어떤(형용사의 관형어)

    - 관형사일때는 선택의 의미, 형용사일 떄는 상태의 의미 

        ex) 어떤 게임 할래? (선택)

        ex) 어떤상태야?


* 보조동사      - 본용언 뒤에 기대어 쓰이면서 본용언에 문법적 의미를 더해준다.

* 보조형용사    - 본용언 뒤에 기대어 쓰이면서 본용언에 문법적 의미를 더해준다.



* 서현이형집에 있다 거기서 방학을 보냈지 

    - 차타고 이동했기때문에 "있다" => 동사

    - "머물다", "시간의 경과"의 "있다"는 동사이다.


* 보조용언에 본래뜻을 살려서 해봤을 때 어색하면 본+보조 관계다.


* 되게 - 파생부사

    - 게(부사파생접미사): 아주, 몹시



* 선어만어미배열순서

    - 시었겠사옵더니

* 'ㅎ' 불규칙

    - 동그랗다  =>  동그래
    - 그렇다    =>  그래
